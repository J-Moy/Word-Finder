About Word Finder

Word Finder is a simple web application that will take any combination of letters and output all of the words that can be built from some or all of those letters. Optionally, preceding or proceeding letters can be given along with the main letters and all words that start or end with the specified letters will be output. This application was made with the intention of making games like Scrabble, Words With Friends, Wordscapes, among others, more easy.

Word Finder is built using Python's Flask framework along with Jinja2 templates and MongoDB for caching past inputs. The Python script itself is written using the Pyenchant and Itertools modules. Pyenchant is a spellchecking module which can also be used to check if a word is indeed an actual English word. Itertools is used for calculating the permutations of the user input letters which are then subsequently checked using Pyenchant to validate whether or not the permutation is a word. The words are then processed and passed to a Jinja2 template for rendering. All this can be seen at the project's GitHub repo.

About Me
